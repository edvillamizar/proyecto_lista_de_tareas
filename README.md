# TaskFlow 📝

Aplicación sencilla de lista de tareas desarrollada con **HTML, CSS y JavaScript Vanilla**.  
Permite crear, completar y eliminar tareas con persistencia en el navegador.

---

## 🚀 Demo

Puedes abrir el proyecto directamente en el navegador ejecutando `index.html`.

---

## 📌 Funcionalidades

- Crear tareas
- Marcar tareas como completadas
- Eliminar tareas
- Contador de tareas pendientes
- Persistencia con LocalStorage (no se pierden al recargar)

---

## 🧠 Tecnologías usadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage
- Manipulación del DOM
